express-server-architecture-boilerplate

express-server-architecture-boilerplate (template)

A simple express server boilerplate built with TypeScript & nodejs.

Installation

Clone this repository to your machine:

git clone https://github.com/jaschahuisman/express-server-architecture-boilerplate.git

Use node package manager npm to install the required node modules:

npm install

Usage

Available scripts are:

# watch (developer debugging)
npm run watch

# test (unit testing)
npm run test

# build
npm run build

# start (start the production)
npm run start

# create documentation
npm run document

Contributing

Pull requests are welcome. For major changes, please open an issue first to discuss what you would like to change.

Please make sure to update tests as appropriate.
